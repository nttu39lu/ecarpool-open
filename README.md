# 🚕 E 起搭：具隱私保護與需求媒合的校園計程車共乘系統

---

## 🧭 專案簡介
E 起搭（E-Carpool）是一套為偏鄉大學校園設計的共乘系統，
提供安全、便利且兼顧隱私保護的計程車共乘服務。

---

## 🔧 系統模組
- 共乘需求管理模組
- 限時假名聊天室模組

---

## 🧠 技術架構
| 類別 | 工具 |
|------|-------------------|
| 前端 | Kotlin, Jetpack Compose |
| 後端 | Firebase (Auth, Firestore, Realtime DB) |
